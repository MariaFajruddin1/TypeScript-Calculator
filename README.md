# TypeScript-Calculator
Typescript cli Calculator

## simple command line calculator using TypeScipt, Node.js 

## Features
* "+ (Addition)",
* "- (Subtraction)",
* "* (Multiplication)",
* "/ (Division)",
* "% (modulus or Reminder)",
* "** (Exponentiation operator / Power)

## Packages:
```
 chalk,
 chalk-animation,
 gradient-string,
 inquirer,
 nanospinner.
```

 ## Setup
```
 npx typescript-calculator
```
